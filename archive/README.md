# Legacy website source archive

This directory preserves the original website capture used during migration.
It is reference material only and is not imported by the Astro application.

- `legacy-site/images.zip` is the original supplied image tree. Its 278 files
  were byte-checked before the redundant extracted copy was removed.
- `legacy-site/manifest.json` records the captured source URLs and local files.

Curated runtime images belong under `public/images/`. Product-image mappings and
classification notes belong under `docs/image-inventory/`.
